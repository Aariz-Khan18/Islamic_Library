# 📚 Islamic Digital Library

A clean, responsive, and ad-free Islamic Digital Library built using pure HTML, CSS, and JavaScript.

This project is developed as a **Sadaqah Jariyah** initiative — completely free forever, with no ads and no monetization.

---

## 🌙 Project Overview

The Islamic Digital Library provides structured access to:

- 📖 Noble Quran (Arabic + Urdu Translations)
- 🎧 Quran Recitation & Translation (YouTube Playlists)
- 📜 Hadith Collections (Kutub al-Sittah)
- 🔍 Real-time search functionality
- 🌗 Dark / Light mode toggle
- 📱 Fully responsive layout
- ⚡ Smooth scroll animations
- ⬆ Back-to-top button

---

## ✨ Key Features

### 🧭 Sticky Navigation
- Smooth scrolling behavior
- Mobile hamburger menu
- Dark mode toggle button

### 🔎 Smart Search
- Real-time card filtering
- Case-insensitive search
- Works across all sections (Quran + Hadith + Intro)

### 🌗 Dark Mode
- Fully styled dark theme
- Persistent visual consistency
- Themed cards, navbar, footer, and accordion

### 📜 Multi-Volume Hadith Accordion
- Expandable `<details>` sections
- Organized volume access
- Clean button-based navigation

### 🎨 UI & Design Highlights
- Islamic geometric subtle background pattern
- Elegant green & gold theme
- Arabic typography using **Amiri**
- Modern typography using **Poppins**
- Hover lift card effect
- Scroll reveal animation using `IntersectionObserver`

## 🛠 Technologies Used

- HTML5
- CSS3 (Flexbox + Responsive Design)
- Vanilla JavaScript
- Google Fonts API
- IntersectionObserver API

No frameworks.  
No external JS libraries.  
Pure frontend implementation.

---

## ⚙ JavaScript Functionalities

- `toggleTheme()` → Dark mode switch
- `toggleMenu()` → Mobile navigation toggle
- `searchLibrary()` → Real-time content filter
- `scrollToTop()` → Smooth scroll to top
- Scroll animation via `IntersectionObserver`

## 🚀 How to Deploy

1. Download or clone repository
2. Place all PDF files in the same directory as `index.html`
3. Open `index.html` in browser
4. (Optional) Deploy via:
   - GitHub Pages
   - Netlify
   - Vercel
   - Any static hosting service
